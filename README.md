🪨 Stone Paper Scissors

This is a simple Stone Paper Scissors game I built using HTML, CSS, and JavaScript.
Nothing too fancy — just a clean little project to practice logic, DOM manipulation, and storing data in the browser.

🚀 Live Demo

https://cyberyash24.github.io/stone-paper-scissors/

---

🎮 What it does

- Lets you play against the computer
- Random CPU choices every round
- Shows result instantly (win / lose / draw)
- Keeps track of your score
- Saves score using localStorage (so it stays even after refresh)
- Reset button to start fresh

---

🛠️ Built with

- HTML
- CSS
- JavaScript
