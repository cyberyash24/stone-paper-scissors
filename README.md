# Stone Paper Scissors Game

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.  
The game keeps track of Wins, Losses, and Draws using localStorage, so your score persists across page reloads.

## How to Play

1. Click one of the three buttons: Rock, Paper, or Scissors.  
2. The CPU will randomly choose its move.  
3. The result (Win, Lose, or Draw) will be displayed, along with updated scores.  
4. Click Reset to clear the scoreboard.
